La communauté des curieux
Nous allons explorer la technologie blockchain ensemble. Ce site web vous permettra d'interagir avec des contrats de la blockchain Ethereum.
Blockchain Privacy By Design
Les informations vous concernant (clés privée et publique) sont stockées sur votre application MetaMask. Les informations concernant les contrats sont stockés sur la blockchain directement. Aucune information personnelle, technique ou fonctionnelle n'est stockée hors de la blockchain afin de garder un esprit de désintermédiation.
L'historique des transaction est stockée sur le navigateur pour simplifier la navigation.

Merci de connecter votre extension/application MetaMask ou de rafraichir la page afin de connecter la wallet. Si cela ne fonctionne pas, fermez et relancez le navigateur ou l'application Metamask.

Merci de suivre cette procédure :
  1. Installer Metamask (application dans le store si vous êtes sur mobile) ou extension Google Chrome si sur desktop
  2. Configurer votre wallet Metamask (avec mot de passe)
  3. Ajouter un nouveau réseau tel que décrit plus haut (avec Add network dans les paramètres puis renseigner dans RPC URL https://curieux.ma/blockchain/ puis 985459 dans chainId)
  4. Accepter la connection au site (une pop-up devrait s'afficher)

Les informations vous concernant (clés privée et publique) sont stockées sur votre navigateur (extension MetaMask) ou sur votre application MetaMask.
Les informations concernant les contrats sont stockés sur la blockchain directement.
Aucune information personnelle, technique ou de nature blockchain n'est stockée sur nos serveurs.
L'historique des transaction est stockée sur le navigateur pour simplifier la navigation. Ce site est simplement un intermédiaire intuitif et user-friendly pour manipuler des smart contracts de manière à respecter l'esprit de la désintermédiation.
